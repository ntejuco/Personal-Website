Ski Field Simulator

The purpose of this assignment was to implement multithreading techniques learnt in class.
Each second a skier object is picked from the front of a ski lift queue implemented using a linked list. A thread is then created and the skier object passed to it, where it is held for a random time within a pre-set range. It is then appended to the end of a waiting line for the ski lift, which is implemented using a second linked list. Upon reaching the front of the queue the skier is appended to the back of the ski lift.
